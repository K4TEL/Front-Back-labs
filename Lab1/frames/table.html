Теорія:

HTML tables allow web developers to arrange data into rows and columns. Each table cell is defined by a <td> and a </td> tag. Each table row starts with a <tr> and end with a </tr> tag. Sometimes you want your cells to be headers, in those cases use the <th> tag instead of the <td> tag
